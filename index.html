Welcome to XiVO auth

Hello, this is your first connection to this system or your xivo auth host connection is empty.

If it your first time, please take time to understand how does it work and what you can do here.

You need to a connection to your XiVO auth service. By default it looks like https://ip_address_of_your_xivo:9497. Please accept the certificate on your service

The backends:

  • XiVO Admin: use your xivo webi user like root.
  • XiVO User: use client cti user.
  • XiVO Service: use a specific user from xivo-manage-key.
  • XiVO WS: use a webservice user set in webi.
